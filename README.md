# Restaurant List
簡單的餐廳推薦系統

## Features
- 表列列出全部餐廳 (source: restaurant.json)
- 提供搜尋功能，可依名稱或類別搜尋喜愛的餐廳
- 點擊喜愛的餐廳可進一步看到詳細資料
